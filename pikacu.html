Pokemon Explorer

A simple Pokédex web app using HTML, CSS, and JavaScript. Users can search for a Pokémon by name and view basic details using data from PokeAPI.

Features:

  • Search Pokémon by name
  • Show name, weight, and image
  • Displays error for invalid names
  • Console log shows full data

Tech Stack:

  • HTML – Page structure
  • CSS – Basic styling (optional)
  • JavaScript – fetch(), async/await, and DOM manipulation
